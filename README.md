# BusinessStart

> Обучающее мобильное приложение для иностранных предпринимателей  
> Помогает освоить процедуры регистрации и ведения бизнеса в Российской Федерации

## 🎯 Функциональность

### 4 обучающих сценария

| Сценарий | Что включает |
|----------|--------------|
| **Регистрация бизнеса** | Выбор адреса, вида деятельности, нотариальная палата (апостиль/консульская легализация), создание устава (права/обязанности), уставной капитал, госпошлина |
| **Постановка на налоговый учёт** | Проверка документов, апостилирование, заполнение формы 11БС-Учет, формирование пакета документов |
| **Открытие счета** | Выбор банков (множественный выбор), просмотр требований каждого банка, заполнение анкеты |
| **Закрытие бизнеса** | Голосование о ликвидации, оповещение госорганов, сбор задолженностей, ликвидационный баланс, расчёты с кредиторами, увольнение сотрудников |

### Модуль тестирования
- Тесты по каждому сценарию
- Автоматическая проверка ответов
- Отображение правильного ответа при ошибке (обучающая обратная связь)
- Подсчёт результата

## 🚀 Демонстрация работы
### Авторизация и регистрация
- Валидация email (формат) и пароля (мин. 6 символов)
- Подтверждение пароля при регистрации
- Отображение/скрытие пароля (иконка глаза)
- Обработка ошибок сервера (409, 422, 400)

### Сценарий «Регистрация бизнеса»
- Динамическая валидация названия (должно содержать «Общество с ограниченной ответственностью»)
- Выбор адреса из справочника (проверка на наличие «офис»)
- Выбор вида деятельности (предупреждение о лицензии)
- Нотариальная палата: выбор страны → определение типа легализации
- Подготовка документов с шагами в зависимости от страны
- Устав: выбор типового/индивидуального, добавление прав/обязанностей
- Уставной капитал (мин. 10 000 ₽) и госпошлина (4 000 ₽)

### Сценарий «Открытие счета»
- Grid-отображение банков с логотипами (Coil, fallback на заглушку)
- Множественный выбор банков (selectedBankIds: Set<Int>)
- Просмотр требований каждого банка (навигация с параметром)
- Имитация заполнения анкеты

### Сценарий «Закрытие бизнеса»
- Голосование участников
- Ликвидационный баланс: активы - пассивы = чистые активы
- Предупреждение о банкротстве при отрицательных активах
- Расчёты с кредиторами (очереди по ст. 64 ГК РФ)

### Тестирование
- Загрузка тестов, вопросов и ответов через UseCase
- Пошаговое прохождение: выбор ответа → проверка → результат
- При неверном ответе показывается правильный ответ
